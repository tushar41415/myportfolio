# Tushar Portfolio

Premium React + TypeScript portfolio site built with Vite, Tailwind CSS, shadcn/ui, and Framer Motion.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

App runs on [http://localhost:8080](http://localhost:8080).

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm run test
```
